# Cybersecurity & IT Fundamentals

This repository covers core cybersecurity and IT fundamentals required for SOC analysts and entry-to-intermediate security roles.

It focuses on practical understanding rather than theory, including:
- Networking and system fundamentals
- Security concepts and attack basics
- Logs, alerts, and investigation thinking
- Identity, authentication, and access fundamentals
- Cloud and third-party dependency basics

The content is built with a blue-team and real-world mindset to help readers understand not just *what* something is, but *why it matters* in real environments.
